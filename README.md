# **Python-Database-Connections**

This repository contains simple and beginner-friendly examples of how to connect Python with different SQL databases.  
It includes hands-on Jupyter Notebooks demonstrating how to:

- Connect to **MySQL** using Python  
- Create tables, insert data, and run queries  
- Work with **SQLite** using Python (built-in database)
- Perform basic CRUD operations (Create, Read, Update, Delete)
