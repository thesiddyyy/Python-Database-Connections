# **Python-Database-Connections**

This repository contains simple and beginner-friendly examples of how to connect Python with different SQL databases.  
It includes hands-on Jupyter Notebooks demonstrating how to:

- Connect to **MySQL** using Python  
- Create tables, insert data, and run queries  
- Work with **SQLite** using Python (built-in database)
- Perform basic CRUD operations (Create, Read, Update, Delete)

---

##  **Repository Structure**



---

##  **MySQL – Python Connection**

Notebook: `mysql/Python_MySQL_Connection.ipynb`  
Contents include:

- Installing MySQL Connector
- Creating a MySQL connection
- Creating tables  
- Inserting & fetching records  
- Closing the connection safely  

---

##  **SQLite – Python Database**

Notebook: `sqlite/SQLite+DB.ipynb`

Contents include:

- Connecting Python to SQLite (no installation needed)  
- Creating and managing tables  
- Inserting and reading data  
- Running SQL queries from Python
